    Deployment

    API

    This section is dedictated to deploying the Democrasales API.

    Staging Instructions:

    Heroku instructions link: https://devcenter.heroku.com/articles/git

    # Setup Heroku Git Remote
heroku git:remote -a democrasales-api-staging

git remote rename heroku staging

    Deployment Setup Instructions:

    • Setup git remote:
      • heroku git:remote -a democrasales-api-staging
    • Rename remote to staging
      • git remote rename heroku staging
    # Staging Deployment Commands
git push staging staging:master

heroku run rake db:migrate --remote staging

heroku restart --remote staging

    Deployment Instructions (Follow these in order when deploying):

    • Deploying staging branch to staging
      • git push staging staging:master
    • Run Migrations:
      • heroku run rake db:migrate --remote staging
    • Restart server:
      • heroku restart --remote staging

    Bootstrap Database

    If using the API locally, seed your database with the questionnaire content:

    $> rake db:seed:questionnaire

    Lead Statuses

    A lead has a lead status and a questionnaire status, each of which may change every time a lead is updated, depending on the state of other lead attributes.

    Status Values
    lead_status follow_up, qualified, disqualified
    questionnaire_status unstarted, started, completed

    Create QuestionAnswer

    This endpoint creates a single question answer.

    HTTP Request

    POST http://democrasales.com/api/v1/question_answers

    Parameter Required Description
    question_id required question ID
    question_option_id required question option id
    value required The serialized string value of answer. The data type is determined by the associated QuestionOption.

    The response looks like this:

    {
  "id": 15,
  "question_id": 1,
  "question_option_id": 4,
  "value": 'true'
}

    Bulk Create QuestionAnswer

    This endpoint creates multiple question answers.

    HTTP Request

    POST http://democrasales.com/api/v1/question_answers/bulk_create

    Parameter Required Description
    question_answers required array of question answers, each containing question_id, question_option_id, and value

    The response looks like this:

    [{
  "id": 15,
  "question_id": 1,
  "question_option_id": 4,
  "value": '6'
},
{
  "id": 16,
  "question_id": 1,
  "question_option_id": 6,
  "value": '2'
}]

    Update QuestionAnswer

    This endpoint updates a single question answer.

    HTTP Request

    PUT http://democrasales.com/api/v1/question_answers/:id

    Parameter Required Description
    id required question answer ID
    value required The serialized string value of answer. The data type is determined by the associated QuestionOption.

    The response looks like this:

    {
  "id": 15,
  "question_id": 1,
  "question_option_id": 4,
  "value": 'true'
}

    Bulk Update QuestionAnswer

    This endpoint updates multiple question answers.

    HTTP Request

    PUT http://democrasales.com/api/v1/question_answers/bulk_update

    Parameter Required Description
    question_answers required array of question answers, each containing id and value

    The response looks like this:
